Package: nz.net.ultraq.thymeleaf.layoutdialect

[Groovy] Class LayoutDialect

        • nz.net.ultraq.thymeleaf.layoutdialect.LayoutDialect 
        • class LayoutDialect
extends AbstractProcessorDialect

          A dialect for Thymeleaf that lets you build layouts and reusable templates in order to improve code reuse.

          To configure the layout dialect, use the fluent API methods: withAutoHeadMerging, withExperimentalTitleTokens, and withSortingStrategy.

          Authors:
          Emanuel Rabina
